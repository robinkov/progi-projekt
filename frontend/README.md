# Getting started

## Clone the repository

```bash
git clone https://github.com/robinkov/progi-projekt.git
cd progi-projekt
```

## Install dependencies

**Only frontend needs local setup, backend is hosted on Render.**
```bash
cd frontend
```
```bash
npm install
```

## Run the application

```bash
npm run web
```
App will be available at `http://localhost:8081`.

Note: Bundling takes a little longer, so be patient on the first run.
